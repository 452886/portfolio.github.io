Run Escape (Godot)

Download here! (Windows)

To complete the trifecta of 'game engine mastery', I decided to make a game with Godot. I had never used Godot before but I was very excited to try it as I am a big advocate for Open-Source software. Furthermore, Godot supports C#, which I was already familiar with.

The result is the game 'Run Escape'! It is a top-down view game wherein the goal is to collect as many coins as possible while avoiding the guard characters that chase you down! I found Godot very great to use and I hope it will catch on in the game industry. Personally, I will definitely keep using it. Perhaps for a Game Jam!
