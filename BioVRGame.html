BioVRCity

This is a project made in collaboration with the Natura Docet museum in Denekamp and Saxion Green office. The goal of the project is to create awareness for the small creatures living in a city environment like ants, bees and worms. The project is a Virtual Reality game in which the user is tasked with managing the environment. Based on what choices the players makes, more or less different creatures will be present in the environment.
